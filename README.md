Raju College Management Idea

This package contains:
 - backend/ : Express + Sequelize (Postgres) minimal API
 - frontend/: React (Create-React-App style minimal) app

Deployment notes:
 - Backend expects POSTGRES_URL or DATABASE_URL in .env (example provided)
 - Frontend uses REACT_APP_API_URL environment variable (see .env.example)
 - You can run backend locally: `npm install` then `npm run start` inside backend/
 - You can run frontend locally: `npm install` then `npm start` inside frontend/

This is a minimal scaffold to help deploy your project. Update models, routes, and env vars as needed.
